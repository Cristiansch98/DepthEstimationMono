KEY FINDINGS / FORMAT
  - MF scenario = 11 s @ 10 Hz = 110 timesteps; first 50 observed, last 60 predicted.
  - parquet is long-format: 1 row per (track, timestep); cols include position_x/y,
    heading, velocity_x/y, object_type, object_category(0-3), focal_track_id, city.
  - track_id == "AV" is the ego/autonomous vehicle. focal_track_id = prediction target.
  - HD map JSON: drivable_areas + lane_segments + pedestrian_crossings.
    lane_segments form a GRAPH (predecessors/successors/neighbors) + boundary
    polylines; centerline is interpolated via get_lane_segment_centerline().
  - MF gives pose+velocity but NOT box size -> used fixed ~4.6x2.0 m footprint.
